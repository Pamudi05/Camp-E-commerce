# Camp E-Commerce (Frontend)

## Overview
A frontend-only e-commerce app to browse products and manage a shopping cart.

## Technologies
- HTML
- CSS
- React
- TypeScript

## Setup
1. Clone the repo: `git clone https://github.com/your-username/camp-ecommerce.git`
2. run project `npm start`
3. instal dependencies `npm i`

## Features
- Home Page
- Product page
- Add to Cart / Place Order
- Responsive design

## Notes
- No backend required
- Cart functionality is frontend-only
